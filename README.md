# Key-Value-Store for bash

## Insert

```bash
kvset key < value
```

## Remove

```bash
kvdel key
```

## Get Value

```bash
kvget key
```

## Contains Key

```bash
kvcontains key
```

## Purge Key-Value Dictionary

```bash
kvclear
```

## Amount of Items in Dictionary

```bash
kvcount
```

## List all Keys in Dictionary

```bash
kvlist
```
